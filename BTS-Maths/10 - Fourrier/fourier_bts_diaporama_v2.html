Fonctions périodiques : rappels

Définition

Une fonction \(f : \mathbb{R} \to \mathbb{R}\) est périodique de période \(T > 0\) si pour tout \(x \in \mathbb{R}\) :

\(f(x + T) = f(x)\)
Pulsation fondamentale
\(\omega_0 = \dfrac{2\pi}{T}\)
Fréquence fondamentale
\(f_0 = \dfrac{1}{T}\)
Condition de Dirichlet (admis)

Une fonction \(f\) de période \(T\) admet un développement en série de Fourier si, sur une période, elle est continue par morceaux et à variation bornée. Toutes les fonctions rencontrées en BTS vérifient ces conditions.

02 / 07

Développement en série de Fourier

Formule générale

Soit \(f\) une fonction \(T\)-périodique vérifiant les conditions de Dirichlet. Son développement en série de Fourier est :

\(f(x) = a_0 + \displaystyle\sum_{n=1}^{+\infty} \bigl[ a_n \cos(n\omega_0 x) + b_n \sin(n\omega_0 x) \bigr] \qquad \omega_0 = \dfrac{2\pi}{T}\)

Les coefficients de Fourier sont :

Valeur moyenne a₀
\(a_0 = \dfrac{1}{T}\displaystyle\int_0^T f(x)\,dx\)
Coefficients aₙ
\(a_n = \dfrac{2}{T}\displaystyle\int_0^T f(x)\cos(n\omega_0 x)\,dx\)
Coefficients bₙ
\(b_n = \dfrac{2}{T}\displaystyle\int_0^T f(x)\sin(n\omega_0 x)\,dx\)
Remarque

L'intégrale peut être calculée sur n'importe quel intervalle de longueur \(T\).

Interprétation de a₀

\(a_0 = \dfrac{1}{T}\displaystyle\int_0^T f(x)\,dx\) est la valeur moyenne de \(f\) sur une période. En électronique : c'est la composante continue du signal.

Impaire : symétrie centrale par rapport à l'origine.

Théorème de Parseval — Valeur efficace

Théorème de Parseval

Si \(f\) est \(T\)-périodique de valeur efficace \(F_{\text{eff}}\) :

\(F_{\text{eff}}^2 = \dfrac{1}{T}\displaystyle\int_0^T [f(x)]^2\,dx = a_0^2 + \dfrac{1}{2}\sum_{n=1}^{+\infty}\bigl(a_n^2 + b_n^2\bigr)\)
Interprétation physique

La puissance totale est la somme des puissances de chaque harmonique.

05 / 07

Spectre en amplitude

Définition

Le spectre en amplitude représente l'amplitude \(C_n = \sqrt{a_n^2 + b_n^2}\) de chaque harmonique en fonction de la fréquence \(f_n = n\,f_0\) (ou de \(n\omega_0\)). C'est un spectre de raies (discret).

Composante continue

\(C_0 = |a_0|\)  →  raie à la fréquence \(f = 0\).

Illustration — Créneau impair d'amplitude \(E\), période \(T\)

Seuls les rangs impairs sont présents : \(\quad C_n = \dfrac{4E}{n\pi}\) pour \(n = 1, 3, 5, \ldots\)

06 / 07

Démarche type pour le contrôle

1

Identifier \(T\) et calculer \(\omega_0 = 2\pi/T\).

2

Tester la parité : paire ⇒ \(b_n = 0\) ; impaire ⇒ \(a_n = 0\).

3

Calculer \(a_0 = \dfrac{1}{T}\displaystyle\int_0^T f(x)\,dx\) (c'est la valeur moyenne).

4

Calculer \(a_n\) et/ou \(b_n\).

5

Écrire le développement jusqu'à l'harmonique demandé.

6

Tracer le spectre : raies aux fréquences \(nf_0\), hauteur \(C_n = \sqrt{a_n^2 + b_n^2}\).

7

Parseval si demandé : \(F_{\text{eff}}^2 = a_0^2 + \dfrac{1}{2}\displaystyle\sum (a_n^2 + b_n^2)\).

Formulaire — intégrales utiles

Intégrale du cosinus
\(\displaystyle\int \cos(n\omega_0 x)\,dx = \dfrac{\sin(n\omega_0 x)}{n\omega_0} + C\)
Intégrale du sinus
\(\displaystyle\int \sin(n\omega_0 x)\,dx = -\dfrac{\cos(n\omega_0 x)}{n\omega_0} + C\)
Valeurs remarquables
\(\sin(n\pi) = 0 \quad \forall\, n \in \mathbb{Z}\)
\(\cos(n\pi) = (-1)^n \quad \forall\, n \in \mathbb{Z}\)
